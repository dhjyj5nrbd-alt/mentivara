Mentivara is an AI-powered tutoring ecosystem for curated private tutors and ambitious students. Not a public marketplace — tutors are recruited and approved by the platform.

**Stack:** React (web, Vite) + React Native (Expo, mobile) + Laravel 11 API + MySQL + Redis
**V1 scope:** Tutor profiles + booking + live classroom (WebRTC)
**No hard constraints** on timeline, budget, or libraries.

**Why:** The platform combines live tutoring, AI study tools, exam prep, community, and mental wellbeing into one ecosystem. V1 focuses on the core tutoring flow before layering AI and community features.

**How to apply:** Always reference PRODUCT_SPEC.md for feature details and PROJECT_PLAN.md for current status. Build incrementally — Phase 1 (MVP) first.
